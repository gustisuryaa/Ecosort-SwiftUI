# 🌱 EcoSort - Swift Student Challenge 2026

EcoSort is a fast-paced, gamified iOS and macOS application designed to educate users on proper waste segregation through rapid visual association.

## ✨ Features
* **Interactive Gameplay:** Sort falling organic and recyclable waste before the timer runs out!
* **Responsive UI:** Built entirely with **SwiftUI** and `GeometryReader` to ensure perfect scaling across iOS and Mac Catalyst.
* **Custom Audio Engine:** Utilizes `AVFoundation` for seamless BGM looping and multi-layered sound effects.
* **Accessibility First:** Features high-contrast colors, clear visual indicators (screen shake on wrong answers), and universal emojis.
* **Micro-Learning:** Displays bite-sized "Eco Facts" upon Game Over to provide actionable real-world knowledge.

## 🛠 Tech Stack
* **Language:** Swift
* **Framework:** SwiftUI
* **Libraries:** AVFoundation
* **Architecture:** State Management (`@State`, `@AppStorage`)
